Dialpad for HubSpot CRM

Dialpad for HubSpot CRM by Sandeza brings cloud telephony directly into HubSpot. Users can make calls, search CRM contacts, manage customer interactions, view call context, and synchronize contact information without leaving HubSpot.

View Screenshots

Overview

The integration embeds the Dialpad softphone directly within HubSpot CRM, allowing sales and support teams to access calling features while working inside their CRM workflow. The application uses HubSpot OAuth authentication, Dialpad APIs, AWS Lambda services, and real-time contact synchronization to deliver a seamless communication experience.

Key Features

☎ Embedded Softphone

Make and manage calls directly from HubSpot without switching applications.

👤 Contact Synchronization

Sync HubSpot contacts and instantly access customer information.

🔍 Smart Search

Search contacts by name or phone number directly from the Dialpad widget.

🔐 OAuth Authentication

Secure HubSpot OAuth integration with token management and account switching.

📞 Active Call Experience

Display customer information and CRM records during live conversations.

📊 CRM Dashboard

Access customer information, tickets, and CRM data from a unified dashboard.

Installation Workflow

Step 1

Install the Dialpad application inside HubSpot CRM.

Step 2

Configure Dialpad credentials and environment settings.

Step 3

Connect HubSpot using OAuth authorization.

Step 4

Select Dialpad Agent and configure CRM settings.

Step 5

Synchronize contacts and start making calls.
